# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- `pyproject.toml` for modern project configuration.
- `CONTRIBUTING.md` guide.
- GitHub Actions CI workflow for testing and linting.
- Pre-commit hooks configuration.
- Logging support replacing print statements in core modules.

### Changed
- Refactored project structure to follow standard Python packaging practices.
- Improved `README.md` with more detailed information and badges.
- Updated `setup.py` to be compatible with `pyproject.toml`.

## [1.0.0] - 2024-05-20

### Added
- Initial release of XML to JSON Converter.
- Support for Brazilian NFe (Electronic Invoice) parsing.
- Batch processing capability.
- CLI interface.
- Basic unit tests.
